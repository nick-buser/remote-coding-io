multi-project PM hub — Linear-esque

Projects are the root: each is its own product with its own features, milestones, and live agent sessions. Features are children of projects; tickets are children of features; sessions spawn from tickets. The sidebar is project-scoped — switch project from the top, browse its features below — and an All projects view shows the full workspace at a glance. Toggle Tweaks to flip accent and density.

Hierarchy
Project ▸ Feature ▸ (PRD, Eng design, Decisions) + Tickets ▸ Sessions
ID system
PRJ-## projects · FEAT-### features · TMX-#### tickets — all auto-assigned
Type
Inter (UI), JetBrains Mono (IDs, code, timestamps)
Accent hues
iris · amber · mint · rose · slate (oklch, fixed C+L, vary H)
Out of scope
Cross-project rollups, multi-stakeholder workflows, branch authoring UI
